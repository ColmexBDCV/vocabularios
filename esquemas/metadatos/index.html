Descripción

Este esquema publica los campos de metadatos utilizados en las plantillas de la Biblioteca Daniel Cosío Villegas para apoyar el mapeo de columnas CSV, hojas de cálculo y otros datos externos hacia términos documentados. Incluye códigos Hyrax, etiquetas, predicados, equivalencias MARC, plantillas aplicables y notas de uso.

Uso para mapeo asistido por IA

Un sistema de IA puede comparar encabezados de un CSV contra los códigos, etiquetas, predicados, MARC, plantillas y notas publicados en los archivos JSON y JSON-LD. Cuando exista ambigüedad, debe preferir coincidencias por código exacto o etiqueta y usar las notas de plantilla para resolver el campo más adecuado.

Campos publicados

200 campos activos o propuestos. La hoja marcada como “no usar” se conserva fuera del esquema activo.
